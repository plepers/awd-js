# awd-js [![Build Status](https://secure.travis-ci.org/plepers/AWD_js.png?branch=master)](http://travis-ci.org/plepers/AWD_js)

AWD tools for javascript

## Getting Started
Install the module with: `npm install awd`

```javascript

var Awd = require('awd');

var awdFile = new Awd();

```
